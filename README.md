# CSGO Demo Parser

node.js modern Source Engine demo file parser. This parser only handles protobuf demos (i.e., Dota 2 and CS:GO).

## Supported Games

- Counter-Strike: Global Offensive

## Supported Features

- [x] Parsing demo files
- [x] User messages
- [x] String tables
- [ ] Entities
- [x] Data tables
- [x] Game events
- [ ] User commands
- [ ] Console commands
